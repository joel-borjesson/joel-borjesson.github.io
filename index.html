CSS Selectors and Specificity Cheatsheet!

Type Selector Specificity Syntax
Basic Selectors Universal top-level, general *
Type More specific than Universal elementname (i.e. table)
Class More specific than Type .classname
ID More specific than Class #idname
Attribute More specific than ID [attr] [attr=value]
Grouping Selector list Groups selectors A, B
Combinators Descendant The " " (space) combinator selects nodes that are descendants of the first element A B
Child The > combinator selects nodes that are direct children of the first element A > B
